前端开源项目推荐

  • UnoCSS

    Unocss是一种基于PurgeCSS的CSS框架,它可以帮助你更简洁、更高效地编写CSS。 Unocss的语法非常简单,它采用了类似于Tailwind

  • Vue3

    易学易用，性能出色，适用场景丰富的 Web 前端框架

  • React

    用于构建 Web 和原生用户界面的库

  • TypeScript

    TypeScript由微软开源的编程语言，它是 JavaScript 的一个超集，而且本质上向这个语言添加了可选的静态类型和基于类的面向对象编程。

  • Vite

    下一代前端开发与构建工具

  • Vitest

    由 Vite 提供支持的极速单元测试框架

  • Pinia

    符合直觉的 Vue.js 状态管理库

  • Ant Design

    一个 UI 设计语言，蚂蚁金服体验技术部出品
